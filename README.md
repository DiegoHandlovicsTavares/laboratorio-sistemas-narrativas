# Laboratório de Sistemas e Narrativas

Este é um espaço onde investigo como sistemas de RPG influenciam a experiência dos jogadores.

Meu objetivo é usar ciência de dados para entender como mecânicas criam sensações como risco, sucesso e narrativa.

## Primeiro experimento

Vou analisar a distribuição de resultados no sistema Fabula Ultima e como isso impacta a experiência de jogo.
